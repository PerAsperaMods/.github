# Organization Wiki - Clean Repository

This directory contains the GitHub Wiki content for the Per Aspera Modding project.

## Files
- `Home.md` - Wiki homepage
- `Getting-Started.md` - Getting started guide
- `Event-System-Quick-Reference.md` - Events quick reference  
- `Event-Classes-Reference.md` - Detailed event classes

## Usage
These files are meant to be synchronized with the GitHub Wiki.

To push to GitHub Wiki (when available):
```bash
git remote add wiki-origin https://github.com/Wafhi3n/PerAsperaMod.wiki.git
git push wiki-origin main
```

## Note
This is a separate git repository to avoid conflicts with the main project.
