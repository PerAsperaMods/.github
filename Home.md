# PerAsperaMod - Per Aspera Modding Framework

🚀 **Central hub for Per Aspera modding!**

## 🎮 About

Per Aspera is a Mars terraforming strategy game. This organization centralizes all resources for creating mods with:
- **BepInEx 6.x IL2CPP** for advanced C# mods  
- **YAML Datamodel** for content modifications
- **GitHub Actions** for CI/CD automation
